### What I learned

I used CSS media query for responsiveness for the first time, though i didnt get it at first but with the help of AI and W3schools i was able to get it right 

I used about CSS Pseudo-elements for the first time and now I can see the useful in incases of changing bullets and numbering colors of unordered and ordered list and at the same time give your text a different color.

Here's the code snippets of the mobile responsivess, see below:

```css
@media screen and (max-width: 480px) {
    body {
        margin: 0;
    }
    main {
        box-sizing: border-box;
        margin: 0;
        width: 100%;
        border-radius: 0;
        padding: 10px;
    }
    img {
        margin-top: 0;
        border-radius: 0;
        margin-left: -10px;
        margin-top: -10px;
        /* margin-right: -10px; */
        display: block;
        width: 100vw;
        padding: 0;
    }
    .prep {
        padding: 20px;
    }
}
```
